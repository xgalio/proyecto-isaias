Proyecto isaias

.

objetivo


Isaias niño de tres años de edad, sufre de una enfermedad llamada esclerosis lateral
amiotrófica la cual ataca ataca a las células nerviosas (neuronas) que están en el cerebro y la médula espinal.
Estas neuronas transmiten mensajes desde el cerebro y la médula espinal hacia los músculos voluntarios, los que usted puede controlar, como los de los brazos y las piernas. Al principio, causa problemas musculares leves.

La Universidad Autonoma De Nariño y nuestro grupo de tranbajo (Ingenieria informatica 5 semestre) queremos ayudar a este niño con la iniciativa llamada PROYECTO ISAIAS en el cual llevaremos una investigacion para realizar una silla que le permita a isaias poder realizar tareas diarias.

¿como lo vamos a hacer?


El eyetracking es una solución tecnológica que pretende extraer información del usuario analizando sus movimientos oculares.
con esta tecnica podremos implementar un software para que el niño pueda intereactuar de manera ocualar
ya sea escribiendo o bien preccionando un boton, este software tambien constara con varias opciones interactivas para facilitar al niño el entendimiento y funcionalidad. implementar tambien una silla de reudas inteligente unida con el software que se puede controlar ya sea a traves de un mando o ya sea por la vista dependiendo de como avanze la enfermedad del niño.

¿que necesitaremos?

Existe varios tipos de software libre para poder llevar a cabo el proyecto entre estos esta :

PyGase: caja de herramientas de código abierto para el seguimiento ocular en Python
entre los dispositivos: silla de reudas controlada por mando para despazarse y sensores conectados al software que permita si la persona va en direccion equivocada o si va a chocar .
